To open the configuration go to Tools->Extensions, then select the synckolab extension and click on options (see image).

  1. Select the address book to sync with (a list of available address books is shown)
  2. Select the email account to sync with (has to be an imap account)
  3. In the list of folders, select an imap folder to sync with
  4. Save to Imap Folder: if not selected the syncronisation will work one way, no data will be changed on the imap folder (no new messages will be added/nothing deleted) the contacts will be changed
  5. Sync Contacts: if not selected, the contact synchronisation will be skipped
  6. Sync Calendar: If not not selected, the calendar (and todo) synchronisation will be skipped
